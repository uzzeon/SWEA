# [SWEA] 2071번. 평균값 구하기

```python
T = int(input())

for test_case in range(1, T + 1):
    numbers = list(map(int, input().split()))
    total = 0
    for number in numbers:
        total += number
    answer = round(total/len(numbers))
    
    print(f'#{test_case} {answer}')
```



- `round()` 반올림
- **f- string**  포매팅 익숙해지기
  - `f'{변수}'`
